**
음수의 나눗셈은 정의하기 나름이라 프로그래밍언어마다 다르게 출력됨
-10 / 3 = ( -3 ) * 3 -1    // 나머지가 음수
-10 / 3 = ( -3 ) * 4 + 2   // 나머지가 양수
c++에서는 나머지가 음수인 경우로 처리하므로 inf_int도 나머지가 음수인 경우로 계산함
즉, inf_int에서 -10 / 3 =  -3 으로 연산함
